## 📏 Core Rule (核心規範)

### 1. Simple SVO (主謂賓)
No change order. 
* *English*: The scripts were updated by me.
* **Loga**: `Wa already update sript.`

### 2. State Clear (狀態明確)
No use word have many mean like "down" or "get".
* `stop`: Shut down / No work
* `wong`: Error / Bug / Fault
* `run`: Doing / Working

### 3. Simple Spell (拼寫簡化)
Short long word. Soft hard sound.
* `Experience` -> **Eberence**
* `Storm` -> **Sidom**
* `Environment` -> **Enviro**
* `Script` -> **Sript**
* `Know` -> **Noi**

### 4. Time & Ask Label (標籤)
Verb always use base form. No plural form for noun. Use label show time and mood.

* **Past**: Add `already` | **Future**: Add `will`
* **Question**: No change order. SVO stay, put question word at end.
    * `U noi what?` (你知道什麼？)
    * `U run sript how?` (你如何運行腳本？)
    * `U like Loga ma?` (你喜歡邏加語嗎？)
    * `Wa will fix wong code.` (我會修復錯誤代碼。)
    * `Wa already fix wong code.` (我已經修復了錯誤代碼。)
